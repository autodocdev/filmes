# Filmes #

Essa aplicação é uma base para um padrão de desenvolvimento Android utilizando Retrofit, Dagger 2 e RxJava.

### Libs ###

* Material Design
* ButterKnife
* Retrofit 2
* OkHttp 3
* Dagger 2
* RxJava
* Glide
* Retrolambda
* RecyclerView Animators
* Image Circle

### Pré Requisitos ###

* Android SDK v16
* Android Build Tools v23.0.1
* Android Support Repository v23.2.1
